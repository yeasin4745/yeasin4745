- 🔍 নতুন টেকনোলজি ও আইডিয়া এক্সপ্লোর করাই আমার অন্যতম প্রিয় কাজ।
- 🧠 ভবিষ্যতে আমি নিজেকে একজন দক্ষ **Web Developer** ও **AI Engineer** হিসেবে গড়ে তুলতে চাই।
